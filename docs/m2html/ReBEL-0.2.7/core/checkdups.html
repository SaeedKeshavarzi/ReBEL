PURPOSE

CHECKDUPS Checks for the presence of duplicate entries in an array

SYNOPSIS

function dups = checkdups(x)

DESCRIPTION 

 CHECKDUPS  Checks for the presence of duplicate entries in an array

     dups = CHECKDUPS(x)

     dups = number_of_duplicates if there are duplicate entries in x
     otherwise dups = 0
