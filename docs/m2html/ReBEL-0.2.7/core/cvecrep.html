PURPOSE

CVECREP Column vector replicate

SYNOPSIS

DESCRIPTION 

 CVECREP  Column vector replicate

   M = cvecrep(V, C) Replicates a Nx1 dimensional column vector V, C times to generate a
   NxC dimensional matrix M.
