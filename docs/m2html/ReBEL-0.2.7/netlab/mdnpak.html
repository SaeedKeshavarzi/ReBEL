PURPOSE

MDNPAK Combines weights and biases into one weights vector.

DESCRIPTION 

MDNPAK    Combines weights and biases into one weights vector.

    Description
    W = MDNPAK(NET) takes a mixture density network data structure NET
    and  combines the network weights into a single row vector W.
